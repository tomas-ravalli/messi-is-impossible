---

## Why this exists

FiveThirtyEight stopped publishing new content in 2024 and its archive has lost all formatting. This repo preserves the full article — text, charts, and images — in a clean, readable form faithful to the original editorial style.

All original images are bundled in the `images/` directory, downloaded from FiveThirtyEight's servers before they go dark.

## Contents

```
index.html          The full article, self-contained
images/             All 18 original charts and photos
README.md           This file
```

## How to publish on GitHub Pages

1. Create a new public GitHub repo (e.g. `messi-is-impossible`)
2. Upload all files in this folder (maintaining the `images/` subfolder)
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)`
5. Save — your site will be live at `https://[your-username].github.io/messi-is-impossible/`
